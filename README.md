# Bandai BB-8

## Components

- Head part
  - ESP32CAM
  - DC Motor(6V 100RPM)
  - ShiftRegister
  - LEDs
  - Custom PCB #1
  - 3D printed Motor Guide
- Body part
  - ESP32(nodeMCU) 
  - ShiftRegister x 2
  - LEDs
  - Custom PCB #2
  - Voltage Step Down module(5v-to-3v) for LEDs
  - dfplayer
- Slip Ring
- VC-02(Voice Recognition module)

## Funtions

- Voice Recognition
- Face Tracking
- Head movements
- LEDs
- Play sounds
