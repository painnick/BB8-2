# Bandai BB-8

## Components

- Head part
  - ESP32CAM + ov2640(Wide)
  - DC Motor(6V 100RPM) + DRV8833 Motor driver
  - ShiftRegister
  - LEDs
  - Custom PCB #1
  - 3D printed Motor Guide
- Body part
  - ESP32(nodeMCU) 
  - ShiftRegister x 2
  - LEDs
  - Hall sensor
  - Custom PCB #2
  - Voltage Step Down module(5v-to-3v) for LEDs
  - dfplayer
- Slip Ring
- VC-02(Voice Recognition module)

## Funtions

- Voice Recognition
- Face Tracking (SoftAP, Android...)
- Head movements
- LEDs
- Play sounds
- Bluetooth Console
